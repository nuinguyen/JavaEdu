Giới thiệu

Ứng dụng web demo Quản lý Khóa học & Đăng ký, xây dựng bằng Spring Boot và Thymeleaf.

Tính năng
  • Đăng nhập / Đăng xuất (JWT, phân quyền Admin / User)
  • Quản lý User (Admin): thêm, sửa, xóa, khóa tài khoản
  • Quản lý Khóa học (Admin): CRUD, sĩ số tối đa
  • Đăng ký khóa học (User): kiểm tra sĩ số, email hợp lệ
  • Duyệt đăng ký (Admin): PENDING → APPROVED
